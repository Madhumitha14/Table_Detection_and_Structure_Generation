# Table Detection and Structure Recognition

## Overview  
This project implements a deep learning pipeline for detecting tables and recognizing column structure in scanned documents. It uses semantic segmentation to generate table and column masks and applies OCR to extract structured text.

---

## Key Features
- Table detection in scanned documents  
- Column structure recognition using segmentation  
- Post-processing for layout refinement  
- OCR-based text extraction and table reconstruction  

---

## Dataset
- Marmot  
- Marmot Extended  

---

## Results
- Table detection accuracy: **98.3%**  
- Column detection accuracy: **98.2%**
